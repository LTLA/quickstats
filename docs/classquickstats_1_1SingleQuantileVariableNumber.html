Detailed Description

template<typename Output_, typename Input_, typename Number_>
class quickstats::SingleQuantileVariableNumber< Output_, Input_, Number_ >

Calculate a quantile from a variable number of elements.

Template Parameters
Output_Floating-point type of the output quantile.
Input_Numeric type of the input values.
Number_Integer type of the objective vector's size.

This uses the same logic as the SingleQuantile class but supports any number of elements, from zero up to a specified maximum.

template<typename Output_ , typename Input_ , typename Number_ >
Output_ quickstats::SingleQuantileVariableNumber< Output_, Input_, Number_ >::operator() ( const Number_ num,
const Number_ nnz,
Input_ * ptr )
inline

Overload to compute the desired quantile from a sparse vector of length num. This vector is assumed to have nnz structural non-zeros and num - nnz zeros.

This method is not thread-safe.

Parameters
numNumber of elements from which to compute the quantile. This should be no greater than max_num.
nnzNumber of elements that are structural non-zero elements. This should be no greater than num.
[in]ptrPointer to the start of an array of length num. This is expected to contain the values of the structural non-zero elements of the sparse vector. It should not contain any NaN values. On output, the order of elements in [ptr, ptr + num) may be rearranged.
Returns
Quantile of the specified probability. If num = 0, NaN is returned instead.
